# Expenses Frontend — Starter Project

This is the starter project for **Exercises 9–11** of the frontend development module.

## What's Included

- `src/services/expenseService.ts` — In-memory CRUD service with hardcoded expense data
- `src/controllers/expenseController.ts` — Controller with methods for all CRUD operations
- `src/index.ts` — Basic Express server listening on port 3000

## Getting Started

```bash
npm install
npm run dev
```

## Your Tasks

Follow the exercises to:

1. **Exercise 9** — Install Nunjucks and GOV.UK Frontend, create a base layout, partials, and read-only pages
2. **Exercise 10** — Add HTML forms for creating, editing, and deleting expenses
3. **Exercise 11** — Replace the in-memory service with Axios calls to your backend API

## Scripts

| Script | Description |
|--------|-------------|
| `npm run dev` | Start the dev server with hot reload |
| `npm run build` | Compile TypeScript to JavaScript |
| `npm start` | Run the compiled JavaScript |
