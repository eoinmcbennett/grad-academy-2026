# Exercise 1 – Create app using prompt engineering

## The task

Recreate the app idea you made in part 1.
This time using the **prompt engineering best practices** that were covered.

- Use the same app idea from part 1
- Use prompt engineering to build on what you learned form part 1
- Try to implement specific features, for example for a timer app:
  - Set or change the timer limit
  - Have preset timer durations
  - Update the styling to match something else you've seen online

> See if you can get the AI to create what you what your want without over-engineering

## Review

Reflect on using prompting engineering concepts.

- Did you find it easier to stay in control of the AI's output?
- Did it still over-engineer?
- Where you able to implement specific changes effectively?
